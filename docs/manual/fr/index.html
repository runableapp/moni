Accueil

The Accueil tab splits the window into two panels. Registre (left) shows results from toolbar buttons and built-in tools — tables, charts, and plain-text output stack in the panel. Use Effacer to reset the stack. Chat (right) talks to the Fournisseur IA you chose under Paramètres → IA; if none is selected, Moni explains how to enable IA.

Toolbar buttons are YAML recipes (see Custom buttons). You can run hledger reports, registers, balance sheets, and external scripts from one click. Chat can suggest hledger commands; optional verification checks them against your installed hledger version.

Comptes

Choose an account from the list (configured under Paramètres → Comptes). Moni shows a balance-over-time chart and a register filtered to that account. Account display names, icons, and institution hints come from your account profiles in Paramètres.

Bénéficiaires

Browse payees extracted from your journal. Select a payee to open its register in a modal dialog — useful for reviewing spending with a single merchant or client.

Rapports

Rapports is a dedicated area for longer-running or summary views — for example monthly P&L-style reports. Content here grows with future releases; many day-to-day reports still live on Accueil via toolbar buttons.

Import

Open Import and drop a .csv, .ssv, or .tsv file on the drop zone, or use the Import ▾ menu. Moni applies your import rules and always previews with hledger import --dry-run before writing to the journal.

Two workflows are available: classic import (full-file import with overlap .latest tracking) and Import intelligent (pick rows, append selected journal text). Toggle Import intelligent under Paramètres → Importation CSV. For the full import guide, open Aide → Import inside the app.

Paramètres

Paramètres is organized into sections along the left: Général, IA, Comptes, MCP, Avancé, Agents IA, Boutons, Apparence, Sauvegarde, hledger, Import, and Journaux. Start with hledger (binary + journal path) and Comptes (which accounts appear on Accueil and Comptes).

See the dedicated Paramètres manual for each section. IA setup is covered in Paramètres → IA.

Aide

The Aide tab ships Markdown topics with the app — English by default, plus localized copies for German, Spanish, French, Japanese, Korean, and Chinese. Topics include import workflows, IA usage, and troubleshooting.

Astuce : Keep a backup of your journal before bulk imports or experimental IA-suggested rewrites. Enable automatic backup under Paramètres → Sauvegarde.
