사용자 가이드

Moni는 일반 텍스트 hledger 저널을 위한 데스크톱 도우미입니다. 이 가이드는 앱 바의 각 탭과 그곳에서 할 수 있는 일을 안내합니다. 아래 스크린샷은 플레이스홀더이므로 준비가 되면 빌드에서 실제 캡처로 교체하세요.

탭 바

창 상단에는 일곱 개의 탭이 있습니다. 이 탭들은 일상적인 원장 작업, 계정 탐색, 보고서, CSV 가져오기, 설정, 앱 내 문서를 함께 다룹니다.

메인

메인 탭은 창을 두 개의 패널로 나눕니다. 원장(왼쪽)는 도구 모음 버튼과 내장 도구의 결과를 표시하며, 표, 차트, 일반 텍스트 출력이 패널에 쌓입니다. 지우기를 사용해 스택을 초기화하세요. 채팅(오른쪽)은 설정 → AI에서 선택한 AI 제공자와 대화합니다. 아무것도 선택하지 않았다면 Moni가 AI를 활성화하는 방법을 설명합니다.

도구 모음 버튼은 YAML 레시피입니다(사용자 지정 버튼 참고). 한 번의 클릭으로 hledger 리포트, 등록부, 대차대조표, 외부 스크립트를 실행할 수 있습니다. 채팅은 hledger 명령을 제안할 수 있으며, 선택적 검증 기능은 설치된 hledger 버전과 대조해 확인합니다.

계정

목록에서 계정을 선택하세요(설정 → 계정에서 설정). Moni는 시간에 따른 잔액 차트와 해당 계정으로 필터링된 등록부를 보여줍니다. 계정 표시 이름, 아이콘, 기관 힌트는 설정의 계정 프로필에서 가져옵니다.

거래처

저널에서 추출한 거래처를 둘러보세요. 거래처를 선택하면 모달 대화상자에서 해당 거래처의 등록부를 열 수 있어, 특정 상점이나 고객과의 지출을 검토할 때 유용합니다.

보고서

보고서는 더 오래 실행되거나 요약 중심의 보기를 위한 전용 영역입니다. 예를 들어 월간 손익 요약 같은 리포트를 볼 수 있습니다. 이 영역의 내용은 앞으로의 릴리스에서 더 늘어날 예정이며, 많은 일상 리포트는 여전히 메인의 도구 모음 버튼에 있습니다.

가져오기

가져오기를 열고 드롭 영역에 .csv, .ssv, 또는 .tsv 파일을 놓거나 가져오기 ▾ 메뉴를 사용하세요. Moni는 가져오기 규칙을 적용하고, 저널에 쓰기 전에 항상 hledger import --dry-run으로 미리보기를 보여줍니다.

두 가지 워크플로를 사용할 수 있습니다. classic import(전체 파일 가져오기와 겹침 .latest 추적)와 스마트 가져오기(행을 선택하고 선택한 저널 텍스트를 추가)입니다. 설정 → CSV 가져오기에서 스마트 가져오기를 전환하세요. 전체 가져오기 가이드는 앱 안에서 도움말 → 가져오기를 여세요.

설정

설정은 왼쪽의 섹션 목록으로 구성됩니다: 일반, AI, 계정, MCP, 고급, AI 에이전트, 버튼, 모양, 백업, hledger, CSV 가져오기, 로그. 먼저 hledger(바이너리 + 저널 경로)와 계정(메인과 계정에 표시할 계정)를 설정하세요.

각 섹션은 전용 설정 설명서에서 확인할 수 있습니다. AI 설정은 설정 → AI에서 다룹니다.

도움말

도움말 탭은 앱과 함께 Markdown 주제를 제공합니다. 기본은 영어이며 독일어, 스페인어, 프랑스어, 일본어, 한국어, 중국어 현지화 사본도 포함됩니다. 주제에는 가져오기 워크플로, AI 사용법, 문제 해결이 포함됩니다.

팁: 대량 가져오기나 실험적인 AI 제안 재작성 전에 저널을 백업해 두세요. 자동 백업은 설정 → 백업에서 활성화할 수 있습니다.
