用户指南

Moni 是面向纯文本 hledger journal 的桌面伴侣。本指南将带你了解应用栏中的每个标签页，以及你可以在其中执行的操作。下面的截图目前是占位图，准备好后请替换为你当前构建中的真实截图。

标签栏

窗口顶部横向排列着七个标签页。它们共同覆盖日常账本工作、账户浏览、报表、CSV 导入、配置以及应用内文档。

标签页用途
🏠 主页账本面板 + AI 聊天；工具栏按钮可运行 hledger 工具
🏦 账户选择账户以查看余额图表和寄存器
🧾 收款方浏览交易方并打开交易方寄存器
📈 报表较长的汇总视图（如月度报表）
📥 导入拖放银行 CSV 导出文件；用 hledger import 预览
⚙️ 设置hledger 路径、AI、按钮、主题、备份等
❓ 帮助应用内置的 Markdown 主题文档

主页面 (主页)

主页 标签页会将窗口分为两个面板。账本（左侧）显示工具栏按钮和内置工具的结果，表格、图表和纯文本输出会堆叠在此面板中。使用 清除 可重置堆叠内容。聊天（右侧）会连接到你在 设置 → AI 中选择的 AI 提供商；如果尚未选择，Moni 会说明如何启用 AI。

工具栏按钮是 YAML 配方（参见 自定义按钮）。你可以一键运行 hledger 报表、寄存器、资产负债表以及外部脚本。聊天 可以建议 hledger 命令；可选的验证功能会将其与你已安装的 hledger 版本进行核对。

账户

从列表中选择一个账户（在 设置 → 账户 中配置）。Moni 会显示余额随时间变化的图表，以及按该账户过滤后的寄存器。账户显示名称、图标和机构提示来自你在 设置 中配置的账户资料。

交易方 (收款方)

浏览从你的 journal 中提取出的交易方。选择某个交易方即可在模态对话框中打开其寄存器，这对于查看与单一商户或客户的支出情况很有帮助。

报表

报表 是一个专门用于较长运行任务或汇总视图的区域，例如月度损益类报表。这里的内容会随着后续版本逐步扩展；许多日常报表目前仍通过 主页 上的工具栏按钮提供。

导入

打开 导入，将 .csv、.ssv 或 .tsv 文件拖到投放区域，或使用 导入 ▾ 菜单。Moni 会应用你的导入规则，并且在写入 journal 之前始终先用 hledger import --dry-run 进行预览。

这里提供两种工作流：classic import（整文件导入并跟踪重叠的 .latest）和 智能导入（选择行并追加所选 journal 文本）。可在 设置 → CSV 导入 中切换 智能导入。完整导入指南请在应用内打开 帮助 → 导入。

设置

设置 左侧按分区组织：常规、AI、账户、MCP、高级、AI 代理、按钮、外观、备份、hledger、导入 和 日志。请先设置 hledger（二进制文件 + journal 路径）和 账户（哪些账户显示在 主页 与 账户 中）。

每个分区的说明请参见专门的 设置 手册。AI 设置详见 设置 → AI。

帮助

帮助 标签页会随应用提供 Markdown 主题文档，默认是英文，同时也包含德语、西班牙语、法语、日语、韩语和中文本地化副本。主题包括导入工作流、AI 用法和故障排查。

提示：在批量导入或尝试 AI 建议的重写之前，请先备份你的 journal。可在 设置 → 备份 中启用自动备份。
