Ajustes

Configuración se abre como una pestaña completa con la lista de secciones a la izquierda y paneles de detalle a la derecha. Configura hledger primero, luego cuentas, botones de la barra e IA opcional. La mayoría de las secciones se guardan en tu carpeta de configuración (~/.config/moni/ en Linux).

General

Idioma de la interfaz, comportamiento al inicio y otras preferencias globales. Cambiar el idioma recarga cadenas localizadas en pestañas, modales y temas de Ayuda donde existan traducciones.

IA

Elige un proveedor de IA para Principal → Chat: llama.cpp u Ollama locales (los datos permanecen en tu máquina), o OpenIA, Claude o Gemini en la nube. El IA Wrapper opcional se conecta al contenedor de escritorio Runable para servicios basados en extensiones.

La IA local incluye descarga de binario/modelo, inicio/parada del servidor, opciones de GPU y RAG de documentación hledger para respuestas fundamentadas.

Guía completa: Configuración → IA

Cuentas

Actualiza el árbol de cuentas desde hledger y elige qué cuentas aparecen en Principal y Cuentas. Añade perfiles de visualización: institución, dígitos enmascarados, iconos y orden. Los perfiles también alimentan transformaciones YAML como merge_account_profiles.

MCP

Configura referencias de servidores Model Context Protocol que Moni puede usar con configuraciones de IA compatibles. Añade definiciones de servidor y detalles de conexión para herramientas que amplían el chat más allá de la ayuda hledger integrada.

Avanzado

Opciones avanzadas: rutas de configuración personalizadas, indicadores experimentales y conmutadores orientados a desarrolladores. Cámbialos solo si comprendes el impacto en escrituras del diario y el comportamiento de la IA.

Agentes de IA

Gestiona definiciones de agentes (prompts de sistema, acceso a herramientas) usadas cuando Moni enruta el chat mediante perfiles de agente. Útil para separar personas de contabilidad y preguntas generales.

Botones

Activa botones de la barra y establece el yamlRef de cada fila a un id de receta o ruta YAML. Reordena filas para cambiar el diseño en Principal. Consulta Botones personalizados (YAML) para crear recetas.

Apariencia

Tema (claro/oscuro/sistema) y tamaño de fuente. El tamaño se aplica a tablas, chat y diálogos de registro de beneficiarios.

Copia de seguridad

Copias de seguridad automáticas del diario antes de que Moni lo modifique. Activa file backup (copia con marca de tiempo junto al diario o en un directorio elegido) y/o git backup (commit en un repositorio). Usa Copia de seguridad now para una instantánea manual; restaura desde archivo o historial git cuando haga falta.

hledger

Obligatorio. Establece la ruta del binario hledger y el archivo principal del diario (-f). Moni ejecuta hledger para saldos, registros, importaciones y recetas de botones. Opciones: mostrar CLI copiable bajo resultados Libro mayor, autocompletado hledger en chat y verificar comandos sugeridos por la IA.

Un asistente de descarga puede obtener una versión precompilada de hledger para tu plataforma si no hay ninguna instalada.

Importararararararararar

Directorio de reglas CSV, asignaciones nombre-de-archivo a reglas, interruptor Importarararararararararación inteligente, dry-run al soltar, comprobación de duplicados, directorio de archivo y comportamiento de copia de solapamiento. Moni resuelve bank.csv.rules primero en el directorio de reglas y luego junto al CSV.

Registros

Moni escribe interaction.log (chat y panel ledger) y app.log (inicio, errores, depuración). Configura el directorio de logs, elige qué eventos registrar, activa depuración en app.log, abre la carpeta o elimina logs. Útil para diagnosticar conectividad de IA o fallos de botones YAML.
