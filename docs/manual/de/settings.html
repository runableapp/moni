Einstellungen

Einstellungen opens as a full tab with a section list on the left and detail panels on the right. Configure hledger first, then accounts, toolbar buttons, and optional KI. Most sections persist to your user config folder (~/.config/moni/ on Linux).

Allgemein

UI language, startup behavior, and other app-wide preferences. Changing language reloads localized strings across tabs, modals, and Hilfe topics where translations exist.

KI

Choose an KI-Anbieter for Übersicht → Chat: local llama.cpp or Ollama (data stays on your machine), or cloud OpenKI, Claude, or Gemini. Optional KI Wrapper connects to the Runable desktop wrapper for extension-based services.

Local KI includes binary/model download, server start/stop, GPU options, and hledger documentation RAG (retrieval-augmented generation) for grounded answers.

Full walkthrough: Einstellungen → KI

Konten

Refresh the account tree from hledger, then choose which accounts appear on Übersicht and Konten. Add display profiles: institution name, masked account hints, icons, and sort order. Profiles also feed YAML transforms like merge_account_profiles.

MCP

Configure Model Context Protocol server references Moni can use with supported KI setups. Add server definitions and connection details for tools that extend chat beyond built-in hledger help.

Erweitert

Power-user options: custom config paths, experimental flags, and developer-oriented toggles. Change these only when you understand the impact on journal writes and KI behavior.

KI-Agenten

Manage agent definitions (system prompts, tool access) used when Moni routes chat through agent profiles. Useful when you want separate personas for bookkeeping vs. general questions.

Schaltflächen

Enable toolbar buttons and set each row’s yamlRef to a recipe id or YAML file path. Reorder rows to change toolbar layout on Übersicht. See Eigene Schaltflächen (YAML) for recipe authoring.

Erscheinungsbild

Theme (light/dark/system) and font size. Font size applies across tables, chat, and payee register dialogs for consistent readability.

Sicherung

Automatic journal backups before Moni changes the journal file. Enable file backup (timestamped copy beside the journal or in a chosen directory) and/or git backup (commit in a repository). Use Jetzt sichern for a manual snapshot; restore from file or git history when needed.

hledger

Required. Set the hledger binary path and main journal file (-f). Moni runs hledger for balances, registers, imports, and button recipes. Optional toggles: show copy-paste CLI under Ledger results, chat hledger autocomplete, and verify KI-suggested commands against your installed version.

A download helper can fetch a prebuilt hledger release for your platform when none is installed.

Import

CSV import rules directory, filename-to-rules mappings, Intelligenter Import toggle, dry-run on drop, duplicate checks, archive directory, and overlap backup behavior. Moni resolves bank.csv.rules from the rules directory first, then beside the CSV file.

Protokolle

Moni writes interaction.log (chat and ledger panel) and app.log (startup, errors, debug). Configure the log directory, choose what interaction events to record, enable debug in app.log, open the folder, or delete logs. Useful when diagnosing KI connectivity or YAML button failures.
