Benutzerhandbuch

Moni ist ein Desktop-Begleiter für hledger-Journale im Klartext. Dieses Handbuch führt durch jede Registerkarte in der App-Leiste und erklärt, was Sie dort tun können. Die Screenshots unten sind Platzhalter — ersetzen Sie sie bei Bedarf durch echte Aufnahmen aus Ihrer Installation.

Die Tab-Leiste

Sieben Registerkarten befinden sich oben im Fenster. Zusammen decken sie tägliche Buchführung, Konten, Berichte, CSV-Import, Konfiguration und integrierte Hilfe ab.

TabZweck
🏠 ÜbersichtLedger-Panel + KI-Chat; Symbolleisten-Schaltflächen starten hledger-Werkzeuge
🏦 KontenKonto für Saldenchart und Kontoblatt wählen
🧾 EmpfängerEmpfänger durchsuchen und Kontoblätter öffnen
📈 BerichteLängere Übersichten (Monatsberichte usw.)
📥 ImportBank-CSV ablegen; Vorschau mit hledger import
⚙️ Einstellungenhledger-Pfade, KI, Schaltflächen, Theme, Sicherung und mehr

Übersicht

Der Übersicht-Tab teilt das Fenster in zwei Panels. Ledger (links) zeigt Ergebnisse von Symbolleisten-Schaltflächen und integrierten Werkzeugen — Tabellen, Diagramme und Klartext stapeln sich im Panel. Mit Löschen setzen Sie den Stapel zurück. Chat (rechts) spricht mit dem unter Einstellungen → KI gewählten Anbieter; ist keiner gewählt, erklärt Moni die Aktivierung.

Symbolleisten-Schaltflächen sind YAML-Rezepte (siehe Eigene Schaltflächen). Mit einem Klick starten Sie hledger-Berichte, Kontoblätter, Bilanzen und externe Skripte. Der Chat kann hledger-Befehle vorschlagen; optionale Prüfung vergleicht sie mit Ihrer installierten Version.

Konten

Wählen Sie ein Konto aus der Liste (unter Einstellungen → Konten). Moni zeigt einen Saldenverlauf und ein gefiltertes Kontoblatt. Anzeigenamen, Symbole und Institutshinweise stammen aus den Kontoprofilen.

Empfänger

Durchsuchen Sie Empfänger aus Ihrem Journal. Wählen Sie einen, um sein Kontoblatt in einem Modal zu öffnen — hilfreich für Ausgaben bei einem Händler oder Kunden.

Berichte

Berichte ist ein Bereich für längere oder zusammenfassende Ansichten — z. B. monatliche GuV-Berichte. Der Inhalt wächst mit künftigen Versionen; viele Alltagsberichte bleiben auf Übersicht über Symbolleisten-Schaltflächen.

Import

Öffnen Sie Import und legen Sie eine .csv, .ssv, or .tsv Datei in die Ablagezone oder nutzen Sie das Import ▾ Menü. Moni wendet Ihre Importregeln an und zeigt immer eine Vorschau mit hledger import --dry-run bevor ins Journal geschrieben wird.

Zwei Workflows stehen bereit: klassischer Import (Gesamtdatei-Import mit .latest Nachverfolgung) und Intelligenter Import (Zeilen wählen, Journaltext anhängen). Intelligenter Import unter Einstellungen → CSV-Import umschalten. Die vollständige Import-Anleitung finden Sie in der App unter Hilfe → Import.

Einstellungen

Einstellungen ist links in Abschnitte gegliedert: Allgemein, KI, Konten, MCP, Erweitert, KI-Agenten, Schaltflächen, Erscheinungsbild, Sicherung, hledger, Import und Protokolle. Beginnen Sie mit hledger (Binary + Journalpfad) und Konten (welche Konten auf Übersicht und Konten erscheinen).

Siehe das Einstellungen-Handbuch für jeden Abschnitt. Die KI-Einrichtung steht in Einstellungen → KI.

Hilfe

Der Hilfe-Tab liefert Markdown-Themen mit der App — standardmäßig Englisch, plus Lokalisierungen für Deutsch, Spanisch, Französisch, Japanisch, Koreanisch und Chinesisch. Themen: Import, KI-Nutzung, Fehlerbehebung.

Tipp: Sichern Sie Ihr Journal vor Massenimporten oder experimentellen KI-Umschreibungen. Automatisches Sicherung unter Einstellungen → Sicherung aktivieren.
