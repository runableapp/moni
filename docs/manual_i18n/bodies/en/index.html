The tab bar

Seven tabs run across the top of the window. Together they cover daily ledger work, account browsing, reporting, CSV import, configuration, and in-app documentation.

🏠 MainLedger panel + AI chat; toolbar buttons run hledger tools

Main

The Main tab splits the window into two panels. Ledger (left) shows results from toolbar buttons and built-in tools — tables, charts, and plain-text output stack in the panel. Use Clear to reset the stack. Chat (right) talks to the AI provider you chose under Settings → AI; if none is selected, Moni explains how to enable AI.

Toolbar buttons are YAML recipes (see Custom buttons). You can run hledger reports, registers, balance sheets, and external scripts from one click. Chat can suggest hledger commands; optional verification checks them against your installed hledger version.

Accounts

Choose an account from the list (configured under Settings → Accounts). Moni shows a balance-over-time chart and a register filtered to that account. Account display names, icons, and institution hints come from your account profiles in Settings.

Payees

Browse payees extracted from your journal. Select a payee to open its register in a modal dialog — useful for reviewing spending with a single merchant or client.

Reports

Reports is a dedicated area for longer-running or summary views — for example monthly P&L-style reports. Content here grows with future releases; many day-to-day reports still live on Main via toolbar buttons.

Import

Open Import and drop a .csv, .ssv, or .tsv file on the drop zone, or use the Import ▾ menu. Moni applies your import rules and always previews with hledger import --dry-run before writing to the journal.

Two workflows are available: classic import (full-file import with overlap .latest tracking) and Smart Import (pick rows, append selected journal text). Toggle Smart Import under Settings → Import. For the full import guide, open Help → Import inside the app.

Settings

Settings is organized into sections along the left: General, AI, Accounts, MCP, Advanced, AI Agents, Buttons, Appearance, Backup, hledger, Import, and Logs. Start with hledger (binary + journal path) and Accounts (which accounts appear on Main and Accounts).

See the dedicated Settings manual for each section. AI setup is covered in Settings → AI.

Help

The Help tab ships Markdown topics with the app — English by default, plus localized copies for German, Spanish, French, Japanese, Korean, and Chinese. Topics include import workflows, AI usage, and troubleshooting.

Tip: Keep a backup of your journal before bulk imports or experimental AI-suggested rewrites. Enable automatic backup under Settings → Backup.
