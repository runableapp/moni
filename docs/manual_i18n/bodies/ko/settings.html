설정은 왼쪽에 섹션 목록, 오른쪽에 상세 패널이 있는 전체 탭으로 열립니다. 먼저 hledger를 설정한 다음 계정, 도구 모음 버튼, 선택적 AI를 구성하세요. 대부분의 섹션은 사용자 설정 폴더(Linux에서는 ~/.config/moni/)에 저장됩니다.

일반

UI 언어, 시작 동작, 기타 앱 전체 기본 설정입니다. 언어를 변경하면 번역이 있는 탭, 모달, 도움말 주제 전반에 걸쳐 현지화된 문자열이 다시 로드됩니다.

AI

메인 → 채팅에 사용할 AI 제공자를 선택합니다: 로컬 llama.cpp 또는 Ollama(데이터가 내 컴퓨터에 머뭅니다), 또는 클라우드 OpenAI, Claude, Gemini. 선택적 AI Wrapper는 확장 기반 서비스용 Runable 데스크톱 래퍼에 연결합니다.

로컬 AI에는 바이너리/모델 다운로드, 서버 시작/중지, GPU 옵션, 근거 있는 답변을 위한 hledger 문서 RAG(retrieval-augmented generation)가 포함됩니다.

자세한 안내: 설정 → AI

계정

hledger에서 계정 트리를 새로 고친 다음 메인과 계정에 표시할 계정을 선택합니다. 기관 이름, 마스킹된 계정 힌트, 아이콘, 정렬 순서를 포함한 표시 프로필도 추가할 수 있습니다. 프로필은 merge_account_profiles 같은 YAML 변환에도 사용됩니다.

MCP

지원되는 AI 설정에서 Moni가 사용할 수 있는 Model Context Protocol 서버 참조를 구성합니다. 내장 hledger 도움말을 넘어 채팅 기능을 확장하는 도구를 위해 서버 정의와 연결 정보를 추가하세요.

고급

고급 사용자를 위한 옵션입니다: 사용자 지정 설정 경로, 실험적 플래그, 개발자용 토글. 저널 쓰기와 AI 동작에 미치는 영향을 이해할 때만 변경하세요.

AI 에이전트

Moni가 채팅을 에이전트 프로필로 라우팅할 때 사용하는 에이전트 정의(시스템 프롬프트, 도구 접근)를 관리합니다. 부기용 질문과 일반 질문에 서로 다른 페르소나를 두고 싶을 때 유용합니다.

버튼

도구 모음 버튼을 활성화하고 각 행의 yamlRef를 레시피 id 또는 YAML 파일 경로로 설정합니다. 행 순서를 바꿔 메인의 도구 모음 배치를 변경할 수 있습니다. 레시피 작성은 사용자 지정 버튼 (YAML)을 참고하세요.

모양

테마(light/dark/system)와 글꼴 크기입니다. 글꼴 크기는 표, 채팅, 거래처 등록부 대화상자 전반에 적용되어 일관된 가독성을 제공합니다.

백업

Moni가 저널 파일을 변경하기 전에 자동으로 저널을 백업합니다. file backup(저널 옆 또는 선택한 디렉터리에 타임스탬프 사본 저장) 및/또는 git backup(저장소에 커밋)을 활성화하세요. 백업 now로 수동 스냅샷을 만들고, 필요할 때 파일 또는 git 기록에서 복원할 수 있습니다.

hledger

필수. hledger 바이너리 경로와 기본 저널 파일(-f)을 설정하세요. Moni는 잔액, 등록부, 가져오기, 버튼 레시피에 hledger를 사용합니다. 선택적 토글: 원장 결과 아래에 복사/붙여넣기용 CLI 표시, 채팅 hledger 자동완성, AI가 제안한 명령을 설치된 버전과 대조해 검증.

설치된 것이 없을 때 플랫폼용 사전 빌드 hledger 릴리스를 가져오는 다운로드 도우미도 사용할 수 있습니다.

가져오기

CSV 가져오기 규칙 디렉터리, 파일명-규칙 매핑, 스마트 가져오기 토글, 드롭 시 dry-run, 중복 검사, 보관 디렉터리, 겹침 백업 동작을 설정합니다. Moni는 먼저 규칙 디렉터리에서 bank.csv.rules를 찾고, 그다음 CSV 파일 옆을 확인합니다.

로그

Moni는 interaction.log(채팅 및 원장 패널)와 app.log(시작, 오류, 디버그)를 기록합니다. 로그 디렉터리 설정, 기록할 상호작용 이벤트 선택, app.log 디버그 활성화, 폴더 열기, 로그 삭제를 할 수 있습니다. AI 연결 문제나 YAML 버튼 실패를 진단할 때 유용합니다.
