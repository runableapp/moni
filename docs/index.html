Foundation

Plain text accounting & hledger

Plain text accounting (PTA) keeps your ledger as human-readable text files — dated transactions and account postings you can open in any editor, back up with git, and automate with scripts. No proprietary database locking you in.

hledger is a fast, dependable command-line tool that reads those journal files and produces balances, registers, and reports. Moni wraps hledger in a modern desktop UI — same data, same power, easier to use every day.

Because everything is plain text and standard CLI commands, hledger integrates cleanly with the rest of your toolchain — grep, git, Python, spreadsheets, cron jobs, and AI scripts can read or transform your journal without a special export step. That power comes with a learning curve: flags, queries, and report syntax take time to memorize.

Moni keeps the same journal and hledger engine, but smooths daily work — one-click custom buttons for repeatable commands, built-in charts for visual summaries, and a fully functional hledger-web-style browse UI for accounts, registers, and reports. Use the GUI when you want speed; drop to the CLI anytime you still need raw power.

Capabilities

CLI power. GUI clarity. Your data.

Moni wraps hledger in a modern desktop app with customizable shortcuts, import tools, and an optional AI copilot — without locking your ledger in someone else's cloud.

⚡

hledger-powered

Balances, accounts, transactions, and reports driven by your configured hledger binary and journal files.

🏠

Local first

Ledgers live as files on your machine. Back up, diff, and version them with the same tools you use for code.

🧠

AI copilot

Ask questions, explain entries, and draft actions. Run local models with llama.cpp or Ollama for privacy — or, if you prefer, connect cloud AI such as OpenAI, Claude, or Gemini.

🔒

Privacy by default

No central SaaS ledger. Cloud AI only when you configure it with your own API keys.

🎛

Custom buttons

Quick-access toolbar shortcuts for your favorite hledger workflows — fully customizable via YAML recipes.

📥

Import & export

CSV import tools and interactive features to review, add, or edit entries with a few clicks.

📈

Reports & registers

Interactive registers, balance sheets, and charts — the full hledger toolkit in a friendly interface.

🛡

Read-only by default

Journal writes are opt-in and audited. Explore safely before you commit changes.

🌐

Multilingual UI

Full interface in seven languages. Help docs and chat can follow your locale.

💻

Cross-platform

Runs natively on Linux, Windows, and macOS — same experience everywhere.
